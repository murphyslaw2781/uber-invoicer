# uber-invoicer
Converts Uber Invoices to CSVs

## Virtual Environment

Set up a virtual environment:
```
python -m venv .venv
```
Activate using:
```
source .venv/Scripts/activate
```

## Install Dependencies

```
pip install streamlit pdfplumber pandas
```

## Run

```
streamlit run main.py
